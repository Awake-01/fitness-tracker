# 健身记录工具

一个功能完善的健身记录与计划管理工具，帮助您轻松追踪训练进度、管理身材数据和饮食记录。

## 功能特点

### 核心功能
- **训练记录**: 支持记录不同类型的训练项目，包括重量训练、计数训练、计时训练和有氧训练
- **训练计划**: 制定和管理个人训练计划
- **身材记录**: 追踪体重、胸围、腰围、臀围等关键身材数据
- **饮食记录**: 记录每日饮食，追踪热量摄入和营养均衡
- **个人档案**: 管理个人基本信息，如年龄、身高、体重、三围等
- **健康计算**: 基于个人数据计算BMI（体重指数）和BMR（基础代谢率）
- **数据管理**: 支持按类型导入/导出或清空特定数据
