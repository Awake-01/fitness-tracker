# 健身记录与计划管理工具

一个功能完整的健身管理Web应用，帮助用户记录训练数据、追踪身材变化、管理饮食摄入，并提供个人健康数据计算。

## 核心功能

### 1. 训练记录与管理
- **多种训练类型支持**：重量训练(kg)、计数训练(个)、计时训练(秒)、有氧训练(分钟)
- **训练项目管理**：可自定义训练项目
- **训练记录**：记录每次训练的项目、组数、重量/次数/时间等数据
- **数据管理**：支持训练记录的导出、导入

### 2. 身材记录
- **多维度数据记录**：体重、胸围（上胸围、下胸围）、腰围、臀围
- **时间线追踪**：按时间顺序展示身材数据变化
- **数据联动**：与健康计算模块联动，自动更新BMI、BMR等健康指标

### 3. 饮食记录
- **饮食时段分类**：早餐、午餐、晚餐、加餐
- **热量记录**：记录食物描述和热量摄入
- **今日统计**：实时显示今日总热量摄入

### 4. 个人档案管理
- **基础信息设置**：性别、出生年月、身高
- **身材数据展示**：显示最新的体重、围度等数据
- **健康指数计算**：
  - **BMI计算**：根据身高体重自动计算体质指数
  - **BMR计算**：根据性别、年龄、身高体重计算基础代谢率
  - **状态评估**：自动评估BMI和BMR的健康状态

## 数据安全

- 所有数据均存储在本地浏览器中，不会上传到任何服务器
- 建议定期使用数据导出功能备份重要数据
- 清除浏览器数据会导致应用数据丢失，请谨慎操作
